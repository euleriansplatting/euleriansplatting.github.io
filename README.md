# Eulerian Gaussian Splatting - Project Website

Research project website for "Eulerian Gaussian Splatting using Hashed Probability Pyramids" (CVPR 2026).

## Project Structure

```
├── index.html              # Main page
├── css/
│   └── style.css          # Global styles
├── js/                    # JavaScript files (if needed)
├── assets/
│   ├── images/           # Project images
│   └── videos/           # Video files
├── README.md             # This file
└── .gitignore           # Git ignore rules
```

## Development

To view the website locally, you can:

1. **Python 3**: `python -m http.server 8000`
2. **Python 2**: `python -m SimpleHTTPServer 8000`
3. **Node.js**: `npx http-server`

Then open `http://localhost:8000` in your browser.

## Customization

- **Update Links**: Replace placeholder links in the hero section with actual paper, code, and arXiv links
- **Add Images**: Place teaser and methodology images in `assets/images/`
- **Add Videos**: Update YouTube embed or upload videos to `assets/videos/`
- **Modify Content**: Edit text sections in `index.html`
